# KAIM Week 3: Insurance Risk Analytics & Predictive Modeling

Analyzing car insurance data for AlphaCare Insurance Solutions (ACIS) to identify low-risk customers and optimize premiums.

## Setup Instructions
1. Clone this repo
2. Run: `python -m venv venv`
3. Activate: `venv\Scripts\activate` (Windows) or `source venv/bin/activate` (Mac/Linux)
4. Install: `pip install -r requirements.txt`
5. Download data: [Week 3 Data Link]

## Branches
- `task-1`: EDA & Stats
- `task-2`: DVC Setup
- `task-3`: Hypothesis Testing
- `task-4`: Predictive Modeling

Raw data stored in `data/` (gitignored per DVC requirements).
